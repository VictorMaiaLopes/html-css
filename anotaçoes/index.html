Anotações importantes

HTML

HTML é conteudo, pode ser em textos, imagens, vídeos, tabelas, listas. Tudo que for focado em conteúdo é HTML.

CSS

CSS é o estilo para o site, focado para deixar o site visivelmente mais bonito: cores, sombras, tamanhos, posicionamento.

Exemplo de estilo em CSS

h1 { font-family: Arial; font-size: 20pt; color: blue; } Com essa tag, você está basicamente dizendo (dando estilo ao seu site) que o seu h1, vai ser na font arial (font-family), o tamanho da fonte vai ser 20pt (font-size), e a cor azul (color:blue)

JavaScript

JavaScript é focado para interações, como criação de: menus, animações, popups, validações.
