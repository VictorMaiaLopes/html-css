Listas ordenadas

Use a tag (OL) para listas ordenadas, e dentro da tag (OL) coloque a tag (LI) para montar a lista, cada item tem que estar dentro de uma tag (LI).

  1. Exemplo
  2. Acordar
  3. Dormir

Você pode alterar o tipo de listas, para números ou letras. Basta apenas dentro da tag (OL) você criar o comando (TYPE). E também o comando (START) para ordenar de qual número ou letra começar.

Listas não ordenadas

São listas que não precisam ser númeradas, sem precisar seguir uma ordem. Basta seguir a tag (UL) e do mesmo jeito da ordenada, é necessario a tag (LI) dentro de cada (UL). Também é possível usar o comando (TYPE) dentro da tag.

Podemos usar os comandos ( TYPE - START )dentro das tags, para ordenar o estilo da marcação, e por qual númeração começar.

Listas de definições

Definition List, use a tag (DL) para criar listas de definições, a tag (DT) é para o termo, e a tag (DD) é para a descrição. As tags (DT - DD) vão dentro da tag (DL).
