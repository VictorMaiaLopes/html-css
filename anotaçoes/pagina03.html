Adicionando imagens

Para adicionar as imagens no nosso site, é importante que as imagens, estejam dentro da pasta de onde você está criando o seu HTML, simplesmente utilizando a tag (img), dentro das "ctrl+espaço", assim o próprio html irá mostrar a pasta com a imagem para você, basta selecionar a certa.
img src="ctrl+espaço" alt="logo HTML5" (alt) serve para explicar o que é a imagem que está sendo carregada.

Também podemos carregar imagens externas; Copie o endereço da imagem e use a mesma tag.
