Anotações importantes

A tag (hr) serve para criar uma linha horizontal (linha deitada)

A tag (br) serve para quebras de linhas, sempre que utilizar (br) a frase após a tag, irá pular uma linha.

Sempre que quiser escrever algum comentário, de alguma parte importante, e que não apareça no site e so usar a tag (< !--)
