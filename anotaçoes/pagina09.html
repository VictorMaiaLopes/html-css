Reproduzindo áudio

Vamos aprender a reproduzir áudio em HTML5.

Existem dois modos para colocar áudios no site. O primeiro que está acima, usando direto a tag (AUDIO), e abaixo, iremos também usar a tag (AUDIO), porém iremos modificala, para que possamos acrescentar mais tipos de midias.

Estilo CSS

No mesmo arquivo, podemos usar CSS externo, CSS interno e o CSS inline. O que fica prevalecendo como configuração final mais forte é o CSS inline, depois o CSS interno e por último o CSS externo.
