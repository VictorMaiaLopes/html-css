Usando links externos

Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal gratuito no YouTube.

A tag (A) chamada de âncora, é possivel adicionar links externos para que você acesse outra página em seu site. Para links externos, e extremamente recomendavel você usar alguns comandos, para que o cliente não saia do seu site, utilizando o comando (TARGET=_BLANK) você consegue fazer com que o link seja aberto em uma nova aba, fazendo com que o seu site continue aberto, assim o usúario irá sempre voltar para seu site. E o comando (REL) selecionando a opção external, irá avisar ao algoritmo, que este link vai para fora do seu site.

Usando links internos

Quando você usa o link que está hospedado no seu próprio computador, é um link interno, então quando adicionarmos ele ao site, não precisamos utilizar os comandos anteriores, porque iremos criar um sistema de navegação em nosso próprio site, e assim não perderemos o cliente.

Clique aqui, para vizualizar a nossa página inicial do site.

Quando o link que você procura, estiver em uma pasta anterior a sua atual, tera que usar (../index.html) < exemplo de como voltar (./index.html) significa a mesma pasta. (../index.html) significa a pasta anterior. Assim sera possível localizar sua página.
